   # 综述

综合利用有标记的数据和没有标记的俄数据，来生成合适的分类函数模型。

   # 方法

生成式半监督学习方法，例如生成式高斯混合模型原理；图半监督学习，等等。

   # 补充

半监督学习的存在，首先要优于单纯只利用存在标记的数据集的结果。与此同时，半监督的目标是提高广泛性，但不是必然会提升的。这源于半监督需要充分利用领域知识来设计模型。
