
   # 综述

通过计算数据特征值之间的距离，根据K值，进行分类。

   # 三要素

K值，距离度量，分类决策规则。 K值通过价差验证平均误差率进行选择，距离度量一般为欧几里得距离，分类决策规则通常采用多数表决法。
